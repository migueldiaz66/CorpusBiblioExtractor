# Options — Most Cited Countries (`mostCitedCountries`)

> Opciones extraídas de biblioshiny 5.4.1 (`inst/biblioshiny/ui.R`, líneas 4891–5019). Ruta de menú: ANALYSIS › Authors › Countries › Most Cited Countries.

## Botones (encabezado)
- **Apply / Run** (`applyMCCountries`)
- **Report** (`reportMCCO`)

## Main Configuration
- **Measure** (dropdown · `CitCountriesMeasure`) — opciones: Total Citations = `TC`, Average Article Citations = `TCY` · default: `TC` (Total Citations).
- **Number of Countries** (entero · `MostCitCountriesK`) — sin min/max/step definidos · default: `10`.

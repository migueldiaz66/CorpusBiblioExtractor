# Options — Sources' Local Impact (`sourceImpact`)

> Opciones extraídas de biblioshiny 5.4.1 (`inst/biblioshiny/ui.R`, líneas 3336–3464). Ruta de menú: ANALYSIS › Sources › Sources' Local Impact.

## Botones (encabezado)
- **Run / Apply** (`applyHsource`)
- **Report** (`reportSI`)

## Main Configuration
- **Impact Measure** (dropdown · `HmeasureSources`) — opciones: H-Index (`h`), G-Index (`g`), M-Index (`m`), Total Citation (`tc`) · default: H-Index (`h`).
- **Number of Sources** (entero · `Hksource`) — opciones: numericInput (sin min/max/step definidos) · default: 10.

# Contributing

Thanks for your interest — but **this is a personal portfolio project — a personal tool built
to ease a bibliometrics thesis development process — not a community project.**

- **Pull requests are not accepted.**
- **Issues are not monitored.**
- There is **no support.**

It is licensed under **GPL-3**. The intended way to build on it is to **fork it**
and take it in your own direction, under the same license.

See [`DISCLAIMER.md`](DISCLAIMER.md) and [`NOTICE.md`](NOTICE.md).
